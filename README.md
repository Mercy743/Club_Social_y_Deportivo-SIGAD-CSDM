# SIGAD-CSDM
Sistema de Gestión para un Club Social y Deportivo

# Descripción
SIGAD-CSDM es un sistema web diseñado para la gestión administrativa y operativa de un club social y deportivo. 
El sistema permite el control de usuarios, socios, horarios deportivos, asistencias, reservaciones, ludoteca y generación de reportes, siguiendo una estructura jerárquica definida.

# Tecnologías
- Aplicación web
- Base de datos relacional
- Control de versiones con Git y GitHub

El proyecto sigue la metodología ágil **Scrum**, organizada en:

- Rama `develop`: integración de funcionalidades
- Ramas `feature/*`: desarrollo de historias de usuario
